1. Файл contact.php - отправка данных с формы на главной странице. Почта указывается в строке #32.
2. Файл sub.php - отправка формы с футера - "Заказ обратного звонка". Почта указывается в строке #10.
3. Путь к файлам PHP необходимо указывать в файле assets/js/main.js в строках #23 и #44 соответственно. Также в этом файле редактируются сообщения после отправки формы.
4. Изменение карты. В каждом файле .html найти строку "<div class="col-md-12 all-map-section">", в ней содержится скрипт Я.Карты, заменить на свой или использовать <iframe> от Google Maps.
